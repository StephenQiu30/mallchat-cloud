# mallchat-log-service - 日志服务

日志服务作为 `mallchat-cloud` 的监控中枢，负责分布式环境下的全链路日志采集、存储及高性能检索，为系统可观测性提供核心支撑。

## 🌟 核心功能

- **分布式日志采集**：
    - 统一入口接收来自各微服务上报的 API 访问日志 (`api_access_log`) 与业务操作日志 (`operation_log`)。
- **高并发写入优化**：
    - 采用异步缓冲机制处理海量日志数据，确保不影响核心业务性能。
- **多维检索支持**：
    - 数据实时同步至 **Elasticsearch**。
    - 提供基于用户 ID、服务名、响应状态及时间范围的高性能组合查询。
- **审计与追踪**：
    - 记录用户登录轨迹、文件上传记录及邮件分发状态。

## 🛠️ 技术栈

- **核心框架**: Spring Boot 3.5.9, MyBatis-Plus
- **搜索引擎**: N/A
- **数据库**: MySQL (基础数据备份)

## 📡 核心 API 概览

| 模块     | 路径                          | 方法   | 描述             |
|:-------|:----------------------------|:-----|:---------------|
| **上报** | `/api/log/inner/save`       | POST | 服务间调用：保存业务日志   |
| **查询** | `/api/log/access/list/page` | POST | 分页检索访问日志 (管理员) |

## 🚀 启动与运行

- **服务端口**: `8086`
- **依赖服务**: Nacos, MySQL
