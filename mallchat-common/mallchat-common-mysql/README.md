# mallchat-common-mysql - 数据库访问基础设施

基于 **MyBatis-Plus** 深度定制，提供全系统的持久化层增强、标准分页策略及核心数据库安全防护。

## 🌟 核心功能

- **MyBatis-Plus 最佳实践**:
    - 自动配置分页插件、乐观锁插件。
    - 统一处理逻辑删除 (`is_delete`) 状态。
- **安全与注入防护**:
    - 提供 `SqlUtils`，对动态排序字段执行严格的正则过滤，杜绝 SQL 注入隐患。
- **自动化字段填充**:
    - 集成 `MetaHandler`，基础表字段（如 `create_time`, `update_time`）无需手动维护，系统自动注入。

## 🛠️ 使用场景

- 适用于所有基于 MySQL 的业务模块，简化持久化层开发，规范数据访问逻辑。
